## Составляющие
Программа реализована на Java и состоит из
- [ColorConverterApp.java](src/main/java/com/shuman/uladzislau/ColorConverterApp.java) -- реализация GUI
- [ColorUtils.java](src/main/java/com/shuman/uladzislau/ColorUtils.java) -- формулы/алгоритмы перевода

## Как запустить
- `git clone git@github.com:UladzislauShuman/computer_graphics.git`
- переходите в папку ~/computer-graphics/lab1
- `mvn clean package`
- `java -jar target/lab1-1.0-SNAPSHOT.jar`
